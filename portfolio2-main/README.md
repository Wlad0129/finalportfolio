# Wladimir Arnedo — Developer Portfolio 🚀

Welcome to the repository for my personal developer portfolio! This project is a modern, responsive, and interactive web application built to showcase my skills as a Frontend Developer & UI Engineer. It highlights my recent projects, professional skills, and even features a special section dedicated to my pets.

## 🌟 Features

- **Modern & Responsive Design**: Built with a mobile-first approach ensuring a seamless experience across all devices (desktops, tablets, and mobile phones).
- **Interactive UI**: Includes dynamic JavaScript functionalities such as:
  - Welcome toast notifications.
  - "Fun Fact" toggles in the About section.
  - Skill list visibility toggles.
  - Smooth scrolling for navigation links.
  - Simulated contact form submission without page reload.
- **Semantic HTML5**: Code structured logically for better accessibility and SEO.
- **Special Pages**: Features an integrated `mascotas.html` page to introduce my favorite animal companions!
- **Fast Performance**: No heavy external libraries or frameworks; just pure, optimized code.

## 🛠️ Tech Stack

This project is built from scratch using the core web technologies:

- **HTML5**: For structure and semantic content.
- **CSS3**: For styling, animations, flexbox/grid layouts, and responsive design.
- **Vanilla JavaScript**: For adding interactivity and managing the DOM.

## 📂 File Structure


## 🚀 Getting Started

Since this is a static website, no complex build tools or servers are required to run it locally.

1. **Clone the repository** (or download the ZIP file):
   ```bash
   git clone https://github.com/yourusername/portfolio2-main.git
   ```
2. **Navigate to the project folder**:
   ```bash
   cd portfolio2-main
   ```
3. **Open the project**:
   Simply double-click the `index.html` file to open it in your default web browser, or use an extension like **Live Server** in VS Code for live reloading.
